# TropiX

**Precision-farming platform for avocado and mango cultivation in the Mediterranean basin.**

TropiX is an Italian agritech startup building the first vertical precision-farming system trained specifically on *Persea americana* and *Mangifera indica* — not adapted from cereal or vineyard data. Founded in Trento, operating in Sicily.

---

## What it does

Three hardware + software layers, delivered as a single annual fee:

| Layer | What's included |
|---|---|
| **Hardware on the farm** | Autonomous multispectral drone (daily flights), per-tree sap-flow sensors, microlocal weather station, edge compute unit |
| **AI intelligence** | Early disease detection (*Phytophthora cinnamomi*, *Colletotrichum gloeosporioides*), per-tree anomaly scoring, flower-count vision, yield forecast ±25% accuracy 6–8 weeks pre-harvest |
| **Marketplace** | Pre-harvest buyer contracts with restaurants and retail, on-demand specialist agronomists, public-incentive tracking |

Key benchmark: **14 days** median lead time on root-rot symptoms before any visible canopy distress.
